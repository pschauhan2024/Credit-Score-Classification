# Credit-Score-Classification
This project focuses on Credit Score Classification using a dataset provided by Paisabazaar, one of India's leading online financial marketplaces. The goal of the project is to build a machine learning model that can accurately classify customers into different credit score categories based on their financial and behavioral attributes

#Dataset: https://drive.google.com/file/d/1P7f1fa0v3hmp5Lc3pbzfVRKOoxY9B66B/view

This project delivered a comprehensive Exploratory Data Analysis on 100,000 customers across 28 financial features spanning 8 months (January through August). The analysis produced 15 meaningful visualizations following the UBM framework: 5 univariate charts exploring distributions and summary statistics, 8 bivariate charts uncovering relationships between features, and 2 multivariate charts including a correlation heatmap and pairplot. The most significant finding was that delayed payments and outstanding debt are the two strongest predictors of poor credit scores and stronger than income alone. The dataset itself was remarkably clean with zero missing values and zero duplicates, though 12,007 placeholder 'NM' values in the payment column required treatment. The colab notebook spans 84 total cells (34 code plus 50 markdown), is fully commented, and runs end-to-end without errors in Google Colab.
